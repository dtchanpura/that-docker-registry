# Simple Docker Registry Frontend

## Building

* Clone the code
* Change parameters in src/main.js like
  * title
  * subtitle
  * app_title
* Install dependencies
  ```bash
  npm install
  ```
* Build code
  ```bash
  npm run build
  ```
* Development live-reload
  ```bash
  npm run dev
  ```
